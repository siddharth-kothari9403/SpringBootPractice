# SpringBootPractice
A repository containing all the applications created as a part of learning spring boot (spring rest api, spring security, hibernate with jpa mysql, spring mvc and thymeleaf, spring data rest, logging and exception handling, junit and mockito)

Credits go to Chad Darby, whose course can be found on Udemy using the following link:
https://www.udemy.com/course/spring-hibernate-tutorial/

Explore California is a training on microservices in Spring Boot, which employs two microservices and builds an API gateway, and integrates with Docker Compose. Credits for the application go to Mary Ellen Bowman, whose course can be found here - 
https://www.linkedin.com/learning/creating-spring-boot-microservices

Landon Hotel application provides a simple usecase using Thymeleaf, Rest Controllers and Repositories. Credits go to Frank P Moley III whose course can be found at - 
https://www.linkedin.com/learning/learning-spring-6-with-spring-boot-3

Setup-junit-intellij provides a review of various techniques used for testing via JUnit. Credits go to Maaike Van Putten whose course can be found at - 
https://www.linkedin.com/learning/java-testing-with-junit-14267963/

Finally, the Organ Management Backend was built as an independent project and employs role based access control using JWT Tokens and Spring Security, with a simple database application which can match donors and recepients.
